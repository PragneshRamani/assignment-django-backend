# assignment-django-backend

## Create virtual env
    - virtualenv venv

## Activate virtual env
    - source venv/bin/activate

## Install Requirements
    - pip install -r requirements.txt

## Migrate
    - python manage.py migrate

## Run server
    - python manage.py runserver

## Server URL
    - http://127.0.0.1:8000/
